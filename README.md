## :wrench: Install and run

Open terminal:

```sh
# Clone this repo
git clone https://github.com/osvaldokalvaitir/reactjs-gostack9-challenge5

# Entry in folder
cd reactjs-gostack9-challenge5

# Install deps with npm or yarn
npm install | yarn

# Launch the app with npm or yarn
npm run start | yarn start
```
